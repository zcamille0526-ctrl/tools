# 📚 教材知识点梳理工具 - 使用说明

## 这个工具做什么？

上传一本教材PDF，工具会自动：
1. 提取PDF文字、识别章节结构
2. 用AI逐章节梳理出知识点
3. 和旧库（九科知识卡）做去重比对
4. 输出一份标准Excel，标注好"已存在"还是"需新增"


## 第一次使用（只需做一次）

### 1. 安装 Python

1. 打开 https://www.python.org/downloads/
2. 点击黄色按钮 **"Download Python 3.x.x"** 下载
3. 运行安装程序
4. ⚠️ **安装界面底部，勾选 "Add Python to PATH"**（非常重要！）
5. 点 "Install Now" 完成安装

### 2. 准备一个大模型 API Key

任选一个，注册后获取 API Key：

## 每次使用

### 启动

双击 **`start.bat`**，等几秒浏览器会自动打开。

如果没自动打开，手动在浏览器输入：http://localhost:8788

### 操作步骤

**第一步 - 上传文件**
- 教材PDF（必须）：要处理的教材，必须是电子版能复制文字的那种
- 旧库表格（必须）：九科知识卡详情的Excel
- 参考表格（可选）：已整理好的样例表格，AI会模仿它的风格

上传PDF后会自动显示章节分割预览，确认章节识别正确再继续。

**第二步 - 配置API**
- 点击预设按钮（通义/DeepSeek/OpenAI）
- 填入你的 API Key
- 选择教材学科（上传旧库后会自动显示可选学科）

**第三步 - 等待处理**
- 处理时间取决于教材长度，一般3-10分钟
- 可以实时看到处理进度和日志

**第四步 - 下载结果**
- 点击"下载Excel"保存

## 常见问题

**Q: 双击 bat 文件闪退？**
A: Python没装好。右键bat文件 → 用记事本打开 → 看看有没有报错信息。或者先打开命令提示符（搜索 cmd），输入 `python --version` 看是否能识别。

**Q: 报错 "pip 不是内部命令"？**
A: 安装Python时没勾选 "Add Python to PATH"。重新安装Python并勾选。

**Q: API报错 401？**
A: API Key 填错了或已过期，去服务商后台重新复制。

**Q: 处理很慢？**
A: 主要看API速度。通义千问一般最快。如果用moonshot-128k，处理长文会更慢些。

**Q: PDF提取出来是乱码？**
A: 说明PDF是扫描件不是电子版。需要先用其他OCR工具处理成可复制文字的PDF。

**Q: 知识点颗粒度不合适？**
A: 上传一份已整理好的"参考表格"，AI会模仿它的颗粒度。


## 安全说明

- API Key 只在你自己电脑上使用，不会上传到任何服务器
- 所有处理都在本机完成
- 关闭命令行窗口即停止服务
